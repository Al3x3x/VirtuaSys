Tipos de Backups

Backup de VM Completa

Copia de seguridad completa de toda la máquina virtual, incluyendo sistema operativo, aplicaciones y datos.

  • Restauración rápida de todo el entorno
  • Ideal para recuperación ante desastres
  • Incluye configuraciones del sistema

Backup de Datos

Respaldo específico de tus datos y archivos importantes, excluyendo el sistema operativo.

  • Menor tamaño y tiempo de respaldo
  • Restauración selectiva de archivos
  • Ideal para respaldos frecuentes

Snapshots

Capturas instantáneas del estado de tu máquina virtual en un momento específico.

  • Creación y restauración instantánea
  • Ideal antes de actualizaciones o cambios
  • No consume espacio adicional

Backup Incremental

Solo respalda los cambios realizados desde el último backup, ahorrando tiempo y espacio.

  • Backups rápidos y eficientes
  • Menor consumo de ancho de banda
  • Ideal para backups diarios

Preguntas Frecuentes sobre Backups

¿Cómo funciona el proceso de backup?

Nuestro sistema realiza una copia de tus datos según la programación establecida. El primer backup es completo, y los siguientes son incrementales para ahorrar tiempo y espacio. Todos los datos se cifran antes de ser transferidos a nuestros centros de datos seguros, donde se almacenan de forma redundante.

¿Cuánto tiempo tarda la restauración de un backup?

El tiempo de restauración depende del tamaño de los datos y el tipo de restauración. La recuperación de archivos individuales suele tardar segundos o minutos, mientras que la restauración de una VM completa puede llevar entre 5 y 30 minutos dependiendo del tamaño. Nuestro sistema está optimizado para minimizar los tiempos de recuperación.

¿Dónde se almacenan mis backups?

Tus backups se almacenan en nuestra infraestructura de almacenamiento distribuida y redundante. Para los planes Profesional y Empresarial, utilizamos replicación geográfica, lo que significa que tus datos se copian en múltiples centros de datos ubicados en diferentes regiones geográficas para garantizar la máxima disponibilidad y protección contra desastres naturales.

¿Puedo acceder a versiones anteriores de mis backups?

Sí, nuestro sistema mantiene un historial de tus backups según el período de retención de tu plan. Puedes acceder y restaurar desde cualquier punto de recuperación disponible a través del panel de control. Esto te permite recuperar datos de hace días, semanas o incluso meses, dependiendo de tu plan.

¿Qué sucede si necesito más almacenamiento para backups?

Puedes ampliar tu capacidad de almacenamiento en cualquier momento desde el panel de control. Ofrecemos paquetes adicionales de almacenamiento que puedes añadir a tu plan actual sin necesidad de cambiarlo. También puedes contactar con nuestro equipo de soporte para obtener una solución personalizada si tus necesidades son muy específicas.
